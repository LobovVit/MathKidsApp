# MathKid

Шаблон SwiftUI-приложения для iPadOS 15 и macOS 11 (Big Sur) для обучения ребёнка:
- сложению
- вычитанию
- умножению
- делению

## Что есть в архиве
- профиль ребёнка
- уровни сложности
- режим ответа: ввод или 4 варианта
- звуки (через `SoundService`)
- детский UI
- родительский экран
- сохранение профиля, настроек и статистики в `UserDefaults`

## Что нужно сделать в Xcode
1. Создать новый SwiftUI App проект.
2. Скопировать файлы из этого архива в проект, сохранив структуру каталогов.
3. Добавить в target папку `Resources/Sounds/` и положить туда реальные файлы:
   - `correct.mp3`
   - `wrong.mp3`
   - `reward.mp3`
4. Убедиться, что все файлы включены в target приложения.

## Минимальные платформы
- iPadOS 15
- macOS 11
